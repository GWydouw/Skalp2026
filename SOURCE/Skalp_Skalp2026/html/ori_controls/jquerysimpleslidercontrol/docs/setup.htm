jQuery Simple Slider Control

Including on a page

To use this script on a page, make sure jquery is included, for example:

<script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.7.2/jquery.min.js"></script>

Then, include the stylesheet and the javascript for the slider itself:

<link rel="stylesheet" type="text/css" href="jscslider.css">
<script type="text/javascript" src="jscslider.js"></script>
(The files jscslider.css and jscslider.js must be uploaded to your server with the pages on which this script will be used.)

Attaching to inputs

Inputs you wish to replace with slider controls should be type="number", and have the min and max attributes set; The field can optionally also have a value set. The input should also have the class "jscslider" assigned to it. The generated slider control should appear as thesame width as the input it replaces.

Additional styles can be applied to each individual slider control by attaching classes to the input with names beginning with "jscslider_" (for example; jscslider_red).

Events and values

When the slider is changed, the value of the attached field is updated. The change() / onchange event is also fired for the input, so additional processing can be performed for this event.
