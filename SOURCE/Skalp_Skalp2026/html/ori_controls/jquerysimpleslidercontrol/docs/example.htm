jQuery Simple Slider Control

Examples

Example showing basic control with a 1 - 100 option, which automatically changes the text in a span:

Current value: 1

HTML Code

<input type="number" min="1" max="100" value="1" style="width:250px;" class="slider" id="id_example_1">
Current value: <span id="id_value_1">1</span>

Javascript code

$('#id_example_1').change(function() {
    $('#id_value_1').text($(this).val());
});

Example used for changing the point size of some text:


Some example text!

HTML Code

<input type="number" min="6" max="48" value="12" class="slider" id="id_example_2"> <br><span id="id_value_2">Some example text!</span>

Javascript code

$('#id_example_2').change(function() {
    $('#id_value_2').css("font-size", $(this).val()+"pt");
});
